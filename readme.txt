Project 1 - Movie trailer Website

What is it?
This project generates a web page to store a list of favorite movies, including box art imagery and a movie trailer URL. Visitors will be able to browse movies and watch the trailers.

Installation
This project does not need any specific installation. Project has been built in Python 2.7.9 framework.  Thus in order for project to run, target machine needs to have 2.7.9 Python framework installed
